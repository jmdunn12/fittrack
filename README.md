# Warrior Fit Tracker

Mobile-first fitness tracker PWA — nutrition targets, a 6-day training split, and bodyweight logging, all in a single self-contained `index.html` (no build step, no backend). Data persists in the browser via `localStorage`.

Add it to your iPhone home screen for a standalone, dark-themed app experience.

Live: https://jmdunn12.github.io/fittrack/
